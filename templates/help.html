Implementation details

This service provides predictions from machine learning models for deciding whether a compound’s protein targets make it likely to be associated with mice longevity.

In the Input page, users can provide examples of compounds and their protein targets, then click the Make Predictions button to get predictions for each entry.

The outputs are predicted percentage likelihoods of being associated with increasing mouse longevity, for each compound, based on their protein targets’ annotations (namely Gene Ontology terms, InterPro protein domains, UniProt keywords and pathway data from Reactome, KEGG and WikiPathways), which are automatically retrieved from the STRING database.

Please see the Help page for a tutorial with examples of how to use the tool, and details about its implementation.

In the Download page, we have made all datasets used to generate the models from this project available for download, as well as a table with all human protein’s STRING code and name, which are used to identify targets in the Input page.
